## Процесс регистрации на сайте

Процесс регистрации на сайте должен включать следующие шаги:

1. Пользователь открывает страницу регистрации.
2. Пользователь заполняет обязательные поля: имя, фамилия, адрес электронной почты, пароль и подтверждение пароля.
3. Система проверяет корректность введенных данных (формат email, совпадение паролей, минимальная длина пароля).
4. Пользователь соглашается с условиями использования (чекбокс).
5. Пользователь нажимает кнопку «Зарегистрироваться».
6. Система отправляет письмо с подтверждением на указанный адрес электронной почты.
7. Пользователь переходит по ссылке из письма для подтверждения аккаунта.
8. После подтверждения аккаунт считается активированным, и пользователь получает доступ к системе.
